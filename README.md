🌿 Lily – Aquaponics Chatbot

Lily is a full-stack chatbot application designed to interact with aquaponics sensor data such as temperature, humidity, water temperature, and pH levels. The system allows users to ask questions in natural language and receive real-time insights based on the latest recorded data.

The backend is built using Python and Flask, where a rule-based intent detection system processes user input and generates responses. Sensor data is read dynamically from CSV files, simulating real-world environmental monitoring. The frontend is implemented with HTML, CSS, and JavaScript, providing a simple and interactive chat interface.

To ensure reliability, the project includes automated testing at multiple levels. Unit tests (Pytest) validate the chatbot logic, while end-to-end UI tests (Playwright) simulate real user interactions in a browser.

This project demonstrates full-stack development, basic NLP concepts, data handling, and automated testing in a real-world scenario.

🛠 Tech Stack
Python (Core Logic)
Flask (Backend API)
HTML / CSS / JavaScript (Frontend)
Pytest (Unit Testing)
Playwright (UI Testing)
